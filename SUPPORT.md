# Support

Split402 is public-alpha protocol infrastructure. It is not production ready,
not mainnet approved, and has no supported release line yet.

No released versions are currently supported.

## Supported Versions

| Version | Support |
| --- | --- |
| Unreleased `main` | Public-alpha development only |
| Published packages | None yet |
| Production/mainnet deployments | Not supported |

## Public Questions

Use GitHub issues for public, non-sensitive questions about:

- protocol schemas, signing, receipt verification, and test vectors;
- x402 extension behavior and local demos;
- SDK integration;
- router and MCP demo gateway behavior;
- documentation gaps.

Do not include private keys, tokens, private URLs, provider credentials, live
transaction bytes, partner details, staging evidence, or custody evidence in
public issues.

## Private Security Reports

Use GitHub Security Advisories for suspected vulnerabilities, including replay
issues, settlement bypasses, payout duplication, auth/session weakness, private
key exposure, or custody concerns.

## Commercial And Hosted Operations

Production hosted services, commercial router operations, provider strategy,
custody operations, private dashboards, and real staging/mainnet evidence are
outside the public support surface unless Split402 explicitly publishes a
sanitized artifact.

## Release Support

No package, container, hosted endpoint, or mainnet deployment should be treated
as supported until it satisfies the release policy in
[`docs/RELEASE_POLICY.md`](docs/RELEASE_POLICY.md).
